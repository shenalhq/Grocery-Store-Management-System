# Grocery Store Management System

A desktop-based Grocery Store Management System developed in **Java Swing** using **Object-Oriented Programming (OOP)** concepts. This application provides an easy way to manage grocery store operations such as product management, inventory handling, and user authentication through a simple graphical interface.

---

##  Project Overview

The Grocery Store Management System is designed to help grocery store staff efficiently manage products and users. The system includes different user roles with separate permissions and uses file handling for data storage.

This project was developed as an academic project to demonstrate Java programming, GUI development, file handling, and OOP concepts.

---

##  Features

###  User Authentication
- User Login
- New User Registration
- Role-based access control

###  Store Manager
- Add new products
- Update product details
- Delete products
- View all products
- Manage inventory

###  Sales Assistant
- View available products
- Search products
- Manage sales-related operations

###  Data Management
- File-based storage
- Read and write product information
- User management

##  Getting Started

### Prerequisites

- Java JDK 8 or above
- NetBeans IDE (Recommended)

### Installation

1. Clone the repository

```bash
git clone https://github.com/shenalhq/Grocery-Store-Management-System.git
```

2. Open the project in NetBeans.

3. Build the project.

4. Run:

```
Main.java
```
